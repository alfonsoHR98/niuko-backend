## Run Locally

Clone the project

```bash
  git clone https://github.com/alfonsoHR98/niuko-backend.git
```

Go to the project directory

```bash
  cd niuko-backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
